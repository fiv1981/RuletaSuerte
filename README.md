# RuletaSuerte

Mini app web tipo Ruleta de la Suerte para 1 jugador.

## Funciones
- ruleta con premios y casillas especiales
- panel con frase oculta
- prueba de letras
- resolver panel
- puntuación y vidas

## Ejecutar
Abre `index.html` en navegador o despliega en Cloudflare Pages o GitHub Pages.
